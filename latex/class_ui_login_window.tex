\doxysubsection{Detailed Description}
A window for user login functionality. 

This class provides the user interface for logging in. It includes text input fields for the username and password, as well as buttons for submitting the login information and navigating to the registration window. The class interacts with the \doxylink{class_authentication_manager}{Authentication\+Manager} to validate the login credentials.
