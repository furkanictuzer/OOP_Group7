\doxysubsection{Detailed Description}
The main window for the application after login. 

Represents the main window of the application.

This class represents the main window that allows the user to navigate between different sections of the application such as expenses, budget, categories, reports, and profile.

This class is responsible for displaying the main window of the application. It includes various buttons to navigate to different sections such as Expenses, \doxylink{class_budget}{Budget}, Categories, Reports, and Profile.
