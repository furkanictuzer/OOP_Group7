\doxysubsection{Detailed Description}
Represents an expense that repeats over a specified interval. 

This class extends the {\ttfamily \doxylink{class_expense}{Expense}} class and adds functionality to handle repeated expenses, including calculating the next occurrence of the expense based on the specified interval.
