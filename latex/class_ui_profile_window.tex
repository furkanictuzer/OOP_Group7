\doxysubsection{Detailed Description}
This class represents the user profile window, where the user can update their username, password, and delete their account. 

A window for managing user profile.

The class provides functionality for displaying the current username, allowing the user to update their username and password, or delete the account. It also provides navigation to the main window or registration window.

This class represents a window that allows users to manage their profile. It provides options to change the username, change the password, delete the account, and navigate back to the main window. It also updates the displayed username after changes.
